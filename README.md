# Hector Rosales Ortiz Portfolio

Premium Next.js 15 personal portfolio for a Senior AI & Full Stack Engineer.

## Stack

- Next.js 15 App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- Three.js, React Three Fiber, React Three Drei
- Material UI
- Lucide React
- Shadcn-style reusable UI primitives

## Scripts

```bash
npm run dev
npm run build
npm run lint
```

## Deployment

The app is Vercel-ready. Push the repository and import it in Vercel with the default Next.js settings.
